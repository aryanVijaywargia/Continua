3. **Air Config for Serve Command**: Updated `.air.toml` to use `full_bin = "./tmp/continua serve"` since `bin` only accepts the binary path, not arguments

4. **SpanTree Build Algorithm**: Used a map-based approach to build tree hierarchy from flat span list, handling missing parents gracefully by treating orphaned spans as roots

### Tree Building Pattern:
```typescript
// Map-based tree construction
const spanMap = new Map<string, SpanNode>();
for (const span of spans) {
  spanMap.set(span.id, { span, children: [] });
}
for (const span of spans) {
  if (span.parent_span_id && spanMap.has(span.parent_span_id)) {
    spanMap.get(span.parent_span_id)!.children.push(spanMap.get(span.id)!);
  } else {
    roots.push(spanMap.get(span.id)!);
  }
}
```
