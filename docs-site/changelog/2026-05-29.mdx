## New features

**Engine runs console.** A new console for inspecting engine runs is now available
in the debugger. You can browse runs, drill into trace detail panes, and trigger
projection repairs directly from the UI. See [Engine runs](/debugger/engine-runs)
and the updated [Trace detail](/debugger/trace-detail) page.

**Seeded engine demo data.** Self-hosted instances can load a curated set of engine
fixtures to explore the console without instrumenting an app first. Useful for
evaluating Continua or onboarding new teammates. See [Self-hosting](/guides/self-hosting).

## Updates

**Public demo projection dry runs.** Projection dry runs can now be executed
against demo projects without an API key, making it easier to share reproducible
debugging scenarios. Authenticated requests are unchanged.

**Refreshed trace detail.** The trace detail view has been reorganized with new
panes for engine context, making it faster to correlate runs with their underlying
traces and spans. See [Trace detail](/debugger/trace-detail).
