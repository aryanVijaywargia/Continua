---
title: "EngineControlClient"
description: "Control-plane client for the Continua durable engine — start, signal, cancel, suspend, resume runs."
---

<Warning>
  `EngineControlClient` targets Continua's **durable engine surface**, which is currently
  scaffolded. Most engine endpoints exist as contracts and partial implementations, not as
  a production runtime. See [Architecture overview](/concepts/overview).
</Warning>

`EngineControlClient` is a thin HTTP client over the engine control-plane endpoints. It
returns Pydantic models from `continua.types` (generated from the OpenAPI contract).

## Constructor

```python
from continua import EngineControlClient

client = EngineControlClient(
    api_key="...",
    endpoint="http://localhost:8080",
    timeout=30.0,
)
```

| Parameter | Default | Description |
| --- | --- | --- |
| `api_key` | _required_ | Engine API key |
| `endpoint` | `"http://localhost:8080"` | Engine server URL |
| `timeout` | `30.0` | Per-request HTTP timeout (seconds) |

Call `client.close()` (or use `with` if supported) to release the underlying HTTP client.

## Lifecycle methods

### `start(request) -> EngineStartRunResponse`

Start a new run. Accepts either a typed `EngineStartRunRequest` or a plain dict.

```python
response = client.start({
    "definition_id": "...",
    "definition_version": 1,
    "input": {"order_id": "abc"},
})
run_id = response.run_id
```

### `signal(run_id, request) -> EngineControlResponse`

Send a signal to a running run.

```python
client.signal(run_id, {"signal_type": "approve", "payload": {"by": "ops"}})
```

### `cancel(run_id) -> EngineControlResponse`

Request cancellation. Best-effort; the engine reaches cancellation on its next quiescence.

### `suspend(run_id) -> EngineRunResponse`

Pause a run. Activities in flight are allowed to complete; no new work is dispatched.

### `resume(run_id) -> EngineRunResponse`

Resume a suspended run.

### `terminate(run_id) -> EngineRunResultResponse`

Terminate a run unconditionally. Returns the final result snapshot.

## Read methods

### `get_instance(instance_key) -> EngineInstanceResponse`

Fetch an instance by its stable instance key.

### `get_run(run_id) -> EngineRunResponse`

Fetch a run's current state.

### `get_result(run_id) -> EngineRunResultResponse`

Fetch a run's terminal result.

### `get_history(run_id, *, page_size=None, page_token=None) -> EngineRunHistoryResponse`

Page through the run's event history.

```python
page = client.get_history(run_id, page_size=100)
events = page.events
while page.next_page_token:
    page = client.get_history(run_id, page_size=100, page_token=page.next_page_token)
    events.extend(page.events)
```

### `get_pending_work(run_id) -> EnginePendingWorkResponse`

Inspect pending activity tasks and timers.

## Maintenance methods

### `purge(run_id, *, force=False) -> EngineControlResponse`

Purge a run from the engine. Forbidden for non-terminal runs unless `force=True`.

### `repair(run_id) -> EngineRepairResponse`

Run repair / reconciliation on a run.

### `backfill_projections(run_id, *, batch_size=None) -> EngineBackfillResponse`

Backfill projections for a single run.

### `backfill_projections_all(*, batch_size=None, page_size=None, dry_run=False) -> EngineBackfillAllResponse`

Backfill projections across all runs. Use `dry_run=True` to preview.

## Polling for terminal state

### `wait_for_terminal(run_id, *, timeout=None, poll_interval=1.0) -> EngineRunResultResponse`

Block until the run reaches a terminal status (`completed`, `failed`, `cancelled`,
`terminated`).

```python
result = client.wait_for_terminal(run_id, timeout=60)
```

Raises:

- `EngineRunWaitTimeoutError` if `timeout` elapses before terminal state
- `EngineRunNotFoundError` if the run does not exist
- `EngineRunContinuationDepthError` if the run exceeds continuation depth limits

See [Exceptions](/sdk/python/exceptions) for the full list.

## Typed responses

All response objects are Pydantic models generated from the OpenAPI contract. Inspect
fields with standard Pydantic APIs:

```python
response = client.get_run(run_id)
response.status        # typed enum
response.model_dump()  # dict for logging
```
